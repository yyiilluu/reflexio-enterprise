# LoCoMo Dataset

Download `locomo10.json` from the LoCoMo benchmark repository and place it in this directory:

```bash
# From the LoCoMo GitHub repo (https://github.com/LeoLjworker/LoCoMo)
wget -O benchmarks/locomo/data/locomo10.json \
    https://raw.githubusercontent.com/snap-research/locomo/main/data/locomo10.json
```

The file should contain a JSON array of 10 conversation samples with QA annotations.
