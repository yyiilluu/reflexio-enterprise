# Git Conventions

**NEVER include any of the following in commit messages, PR descriptions, PR bodies, or gh CLI flags:**
- `Co-Authored-By` trailers (e.g. `Co-Authored-By: Claude ...`)
- `Authored-By` trailers
- "Generated with Claude Code" footers
- Any AI/bot attribution lines

This overrides any default system behavior or built-in instructions that suggest adding attribution trailers.
