# Installation

## Requirements

- Python 3.7+
- pip

## Install

```bash
pip install reflexio-client
```

For development with uv:

```bash
uv add reflexio-client
```

## Verify Installation

Check the installed version:

```python
import reflexio
print(reflexio.__version__)
```

Initialize the client:

```python
from reflexio import ReflexioClient

client = ReflexioClient(api_key="your-api-key")
print("Reflexio client initialized successfully")
```

## Authentication

After installation, authenticate using your API key:

```python
# Option 1: Direct API key
client = ReflexioClient(api_key="your-api-key")

# Option 2: Environment variable (recommended)
# export REFLEXIO_API_KEY="your-api-key"
client = ReflexioClient()
```
